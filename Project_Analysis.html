save_plot <- function(plot_obj, file_name, width = 8, height = 6, dpi = 300, format = "png") {
  # Ensure the format is in lowercase for case insensitivity
  format <- tolower(format)
  
  # Define the plot file extension based on the format
  if (format == "png") {
    # Save as PNG
    png(paste0(file_name, ".png"), width = width, height = height, units = "in", res = dpi)
  } else if (format == "pdf") {
    # Save as PDF
    pdf(paste0(file_name, ".pdf"), width = width, height = height)
  } else if (format == "jpeg") {
    # Save as JPEG
    jpeg(paste0(file_name, ".jpeg"), width = width, height = height, units = "in", res = dpi)
  } else {
    stop("Unsupported file format")
  }
  
  # Print the plot to the file
  print(plot_obj)
  
  # Close the device
  dev.off()
  
  print(plot_obj)
}

Based on the results of the linear mixed effects models it appears that there is a statistically significant influence of woodland treatment on invertebrate populations in some contexts. For example, pitfall invertebrate richness was significantly negatively correlated in Unforested Heathland (p = 0.0426) with Regenerating woodland being nearly significantly negatively correlated (p = 0.0768). In the soil invertebrate models regenerating woodland, unforested heathland and unforested remnant woodland were all statistically significantly correlated with a reduced richness (p = 0.0115; 0.0069; 0.0011) respectively.

The implication here is that mature stands harbour a higher invertebrate richness than unforested treatments or regenerating woodland. This may be due to the prevalence of woodland specialists as there was no influence of woodland treatment on diversity / evenness.

Considering the results for both invertebrate and plant communities, it appears that woodland treatment has an influence on community richness moreso than evenness. In the case of natural regeneration, it appears to either induce a higher plant community richness or the seedlings are preferentially establishing in environments with a higher richness.

This issue canâ€™t be resolved by considering seedling numbers in Mature woodland due to the bias introduced by proximity to reproducing trees. Unforested treatments couldnâ€™t be used either as a methodological bias exists due to these plots being selected for low seedling levels. To resolve the problem would require an experimental set up where pre-seedling community richness of a series of unforested plots was taken and then all exposed to equal levels of natural regeneration. If the pre-seedling richness landscape correlated with the degree of seedling establishment then we could make the conclusion that seedlings were preferentially establishing on higher richness plant communities. If the opposite were true, there would be grounds to suggest that seedling establishment was driving the increase in richness. To test this, seedlings could be inserted into controlled unforested heathland plots and have their local plant communityâ€™s richness tracked with the theory that plots where seedlings establish would increase in richness as a result.
